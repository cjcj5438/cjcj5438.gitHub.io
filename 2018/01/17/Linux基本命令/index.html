Linux基本命令

Linux 基本命令

  1. ls 查看文件目录
  2. dir 查看文件目录
  3. ls -l 查看文件详情
  4. ls -a 查看隐藏文件 ==.== 当前目录; ==..== 上一级目录
  5. cd 切换目录
  6. mkdir xxx 创建xxx目录
  7. cp <目标文件> [期望文件路径]/<重命名文件名> eg:cp oldfile C/newfile 复制文件
  8. cd ../.. 退回2级目录
  9. cp ==-R== oldfile newfile 复制目录: 在当前目录下复制oldfile目录 同时改名newfile
  10. pwd 显示当前用户所有目录
  11. rm <目标文件> 删除文件
  12. rm ==-r== <目标文件>
